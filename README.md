## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Drian27/buiten-tech.git
   cd buiten-tech

   ```

2. Install dependencies:

   ```bash
   npm install

   ```

3. Run the development server:

   ```bash
   npm run build
   npm run dev
   ```
